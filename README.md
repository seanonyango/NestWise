# NestWise – Personal Finance Tracker App

**NestWise** is a mobile app designed to help users manage their spending, track expenses, and save towards personal financial goals.  
Built with **Kotlin** and **Jetpack Compose**, the app focuses on consistency in everyday budgeting.

---

## 🚀 Features (Planned)
- Expense tracking and categorization
- Budgeting and savings goal management
- CSV import and sync for transactions
- Visual spending insights and reminders

---

## 🧱 Tech Stack
- **Language:** Kotlin
- **Framework:** Jetpack Compose
- **Database:** Room (local persistence)
- **Architecture:** MVVM
- **Version Control:** Git + GitHub

---
